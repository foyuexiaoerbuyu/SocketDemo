# SocketDemo

## 具备功能

1. 服务端和客户端在同一个局域网中进行通讯。
2. 支持文字消息和表情消息。

## 使用过程

1. 同一个WIFI网络下，两台Android手机连接此WIFI。
2. 一台手机作为服务端，另一台手机作为客户端。
3. 服务端页面中可以看到IP地址，然后开启服务，等待客户端连接。
4. 客户端连接服务，在弹窗中输入服务端的Ip地址（Ip地址会根据不同的WIFI网络而变化）。
5. 任意一端发送消息都可以，另一端收到后会显示。
6. 可以发送表情，或者表情 + 文字。
